# Omnicon – Universal Calculator Web App
#live Demo
https://omni-nm3836hsd-ovikizhimomi11s-projects.vercel.app/

Omnicon is a **modern all-in-one universal calculator** built using **HTML, CSS, and Vanilla JavaScript**.
It combines **basic calculations, scientific functions, unit conversions, and global currency conversion** in a single responsive web application.

The goal of Omnicon is to provide a **clean, powerful, and customizable calculation tool** that runs entirely in the browser without external frameworks.

---

## Features

### 1. Basic Calculator

* Addition, subtraction, multiplication, division
* Percentage calculations
* Parentheses support
* Decimal support
* Memory functions (MC, MR, M+, M−)
* Calculation history panel

### 2. Scientific Calculator

Advanced mathematical operations including:

* sin, cos, tan
* asin, acos, atan
* log, ln
* square root and cube root
* power functions (xʸ)
* factorial
* π and e constants
* degree / radian toggle
* exponent notation
* absolute value

### 3. Universal Unit Converter

Supports multiple unit categories:

* Length
* Mass / Weight
* Temperature
* Area
* Volume
* Speed
* Time
* Data size
* Energy
* Pressure
* Angle

Users can quickly switch between units using dropdown menus.

### 4. Global Currency Converter

* Supports **150+ world currencies**
* Includes common currencies such as USD, EUR, GBP, INR, JPY, CNY, AUD, CAD, CHF, SGD, HKD, NZD, RUB, BRL, ZAR, KRW, MXN and more
* Real-time conversion using a currency exchange API
* Automatic rate updates
* Swap currency button for quick conversion

If the API is unavailable, the application falls back to **mock exchange rates** to maintain functionality.

### 5. Modern User Interface

* Unified calculator interface
* Mode switching tabs:

  * Basic
  * Scientific
  * Converter
  * Currency
* Large responsive display
* Mobile-friendly layout
* Smooth animations
* Calculation history panel

The design follows a **modern glassmorphism style** inspired by smartphone calculators.

### 6. Theme Customization

Users can personalize the calculator interface with:

* Background color
* Button color
* Accent color
* Display color

Additional options:

* Dark mode
* Light mode
* Reset theme button

Themes update instantly using **CSS variables**.

### 7. JavaScript Architecture

The project uses modular JavaScript components:

* Calculator Engine
* Scientific Function Module
* Unit Converter Engine
* Currency API Module
* Theme Controller
* History Manager

Other features:

* Keyboard input support
* Input validation
* Error handling



## Technologies Used

* HTML5
* CSS3
* Vanilla JavaScript
* CSS Variables
* Browser Fetch API (for currency rates)

No external frameworks or libraries are required.

## Usage

1. Select the desired mode using the tabs:

   * Basic
   * Scientific
   * Converter
   * Currency

2. Perform calculations or conversions.

3. Customize the theme using the **settings panel**.

4. View previous calculations in the **history panel**.
